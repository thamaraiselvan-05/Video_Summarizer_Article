# 🎥 AI Video Insight Generator

Convert YouTube videos into structured articles and downloadable PDFs using Generative AI.

## 🚀 Features
- Extract transcript from YouTube
- AI-powered summarization (Gemini)
- Structured article generation
- Key insights extraction
- PDF export
- Streamlit web interface

## 🛠 Tech Stack
- Python
- Streamlit
- Google Gemini API
- NLP
